# Sakuku - Smart Financial Assistant for Students 

**Sakuku** adalah aplikasi manajemen keuangan pribadi berbasis Android yang dirancang khusus untuk mahasiswa. Aplikasi ini membantu pengguna mengelola anggaran bulanan dengan pendekatan yang lebih intuitif dan edukatif melalui fitur unik seperti **Indomie Index** dan **Limit Harian Dinamis**.

## Fitur Unggulan

- **Limit Harian Dinamis (LHD):** Sistem otomatis menghitung jatah belanja harian yang aman berdasarkan sisa budget dan sisa hari dalam bulan tersebut. Jika hari ini boros, jatah esok hari akan otomatis berkurang.
- **Indomie Index:** Visualisasi saldo sisa budget ke dalam satuan "Bungkus Indomie", memberikan gambaran nilai uang yang lebih nyata bagi mahasiswa.
- **Manajemen Transaksi:** Mencatat pemasukan dan pengeluaran dengan cepat dan mudah.
- **Manajemen Kategori:** Tersedia kategori default dan kemampuan bagi pengguna untuk menambah kategori kustom sesuai kebutuhan.
- **Laporan Visual:** Analisis pengeluaran dalam bentuk **Pie Chart** dengan filter rentang waktu (Mingguan/Bulanan).
- **Smart Notification:** Peringatan visual (warna indikator) jika pengeluaran harian melebihi jatah yang ditentukan.
- **Keamanan Data:** Data disimpan secara lokal di perangkat menggunakan **Room Database**, menjamin privasi dan akses tanpa internet (Offline-First).
